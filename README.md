# CommunityFirst Healthcare Consulting Website

This is a simple one-page Next.js website for CommunityFirst Healthcare Consulting.

## Files included

- `package.json`
- `app/page.js`
- `app/layout.js`
- `app/globals.css`
- `tailwind.config.js`
- `postcss.config.js`

## Deploying

Upload these files and folders to your GitHub repository. Then import the repository into Vercel and deploy it as a Next.js project.
